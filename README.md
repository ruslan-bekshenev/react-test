# Тестирование

Цель тестирования - проверка соответствия ПО предъявляемым требованиям.

## Виды тестов

Функциональное тестирование:

- Модульное (unit)
- Интеграционное
- end-to-end(e2e)

Нефункциональное тестирование:

- Нагрузочное тестирование
- Тестирование безопасности
- Регрессионное тестирование
